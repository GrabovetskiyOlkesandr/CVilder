Як правильно вводити вакансію

Для максимально точного результату система шукає в тексті вакансії конкретні блоки. Не вводьте просто одне слово (наприклад, "Python"). Копіюйте повний текст вакансії.

Обов'язкові ключові слова-заголовки:
  • Вимоги: Requirements, Required, Вимоги
  • Обов'язки: Responsibilities, Обов'язки
  • Буде плюсом (необов'язково): Plus, Nice to have, Буде плюсом
Приклад правильного запиту:

Requirements:
- Python, Django, SQL
- Git, Docker

Responsibilities:
- Backend development
- API integration

Як працює алгоритм оцінки

  • Система відфільтровує граматичне сміття, міста та загальні ІТ-фрази.
  • Вимоги (Requirements) мають найвищу вагу при розрахунку відсотка.
  • Обов'язки (Responsibilities) мають середню вагу.
  • Додаткові навички (Plus) мають найменшу вагу.
